## 1. 系统概述

### 1.1 系统简介

PPTist 是一个基于 Web 的在线演示文稿应用，提供完整的幻灯片编辑和演示功能。系统支持多种常用元素类型，包括文字、图片、形状、线条、图表、表格、视频、音频和公式，并提供丰富的编辑和演示工具。

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      PPTist 应用架构                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐   │
│  │   编辑器    │    │   播放器    │    │  移动端     │   │
│  │  (Editor)   │    │  (Screen)   │    │  (Mobile)   │   │
│  └─────────────┘    └─────────────┘    └─────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    公共模块                           │   │
│  │  历史记录 | 快捷键 | 右键菜单 | 剪贴板 | 主题       │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 核心设计理念

- **数据驱动**：所有编辑操作都体现为数据的修改
- **所见即所得**：画布编辑与最终展示效果一致
- **操作便捷**：丰富的快捷键和右键菜单支持
- **渐进增强**：从基础编辑到高级演示的完整功能链

## 3. 基础编辑功能

### 3.1 历史记录

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 撤销 | 撤销上一步操作 | Ctrl+Z / ⌘+Z |
| 重做 | 恢复被撤销的操作 | Ctrl+Shift+Z / ⌘+Shift+Z |

**技术实现**：
- 使用 ProseMirror 的 history 插件
- 快照存储在 SnapshotStore 中
- 支持无限次数的撤销/重做（受限于内存）

### 3.2 快捷键系统

项目支持几十种快捷键，覆盖常用操作。

#### 3.2.1 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+X | 剪切 |
| Ctrl+A | 全选 |
| Delete | 删除选中 |
| Ctrl+D | 复制并粘贴 |
| Ctrl+G | 组合元素 |
| Ctrl+Shift+G | 取消组合 |

#### 3.2.2 视图控制

| 快捷键 | 功能 |
|--------|------|
| Ctrl+滚轮 | 缩放画布 |
| 空格+拖拽 | 移动画布 |
| Ctrl+0 | 适应画布 |
| Ctrl+1 | 100% 缩放 |
| Ctrl+2 | 适应宽度 |

#### 3.2.3 元素操作

| 快捷键 | 功能 |
|--------|------|
| 方向键 | 移动选中元素（1px） |
| Shift+方向键 | 移动选中元素（10px） |
| Ctrl+] | 上移一层 |
| Ctrl+[ | 下移一层 |
| Ctrl+Shift+] | 置于顶层 |
| Ctrl+Shift+[ | 置于底层 |
| Ctrl+L | 锁定/解锁元素 |

### 3.3 右键菜单

系统提供上下文相关的右键菜单，支持快速操作。

#### 3.3.1 画布右键菜单

- 插入页面
- 网格线设置
- 标尺设置

#### 3.3.2 元素右键菜单

- 复制/粘贴

- 组合/取消组合

- 超链接设置

## 4. 幻灯片页面管理

### 4.1 页面操作

| 功能 | 说明 |
|------|------|
| 添加页面 | 在当前页前后插入新幻灯片 |
| 删除页面 | 删除选中的幻灯片 |
| 复制页面 | 复制当前幻灯片 |
| 粘贴页面 | 粘贴复制的幻灯片 |
| 移动页面 | 拖拽调整幻灯片顺序 |

### 4.2 页面设置

#### 4.2.1 背景设置

支持三种背景类型：

| 类型 | 说明 | 配置项 |
|------|------|--------|
| 纯色 | 单色背景 | 颜色选择 |
| 渐变 | 双色渐变 | 起始色、结束色、角度 |
| 图片 | 图片背景 | 图片源、填充方式（cover/contain/repeat） |

#### 4.2.2 页面动画

| 动画类型 | 说明 |
|----------|------|
| 切换动画 | 页面切换时的过渡效果 |
| 元素动画 | 页面内元素的入场、退场、强调效果 |

**切换动画效果**：
- 无（No）
- 淡入淡出（Fade）
- 水平滑动（Slide X）
- 垂直滑动（Slide Y）
- 随机（Random）
- 3D 水平滑动（Slide X 3D）
- 3D 垂直滑动（Slide Y 3D）
- 旋转（Rotate）
- 缩放（Scale）

### 4.3 幻灯片分节

支持将幻灯片分组管理：

- 创建新节
- 删除节
- 重命名节
- 移动节内幻灯片
- 折叠/展开节

### 4.4 模板功能

提供预设幻灯片模板：

- 封面页模板
- 目录页模板
- 内容页模板
- 结束页模板

用户可以：
- 从模板创建幻灯片
- 将当前幻灯片保存为模板
- 提取已有幻灯片风格

## 10. 用户操作流程

### 10.1 创建新幻灯片

```
1. 打开应用
2. 点击"新建"或使用空白模板
3. 进入编辑器界面
```

### 10.2 编辑幻灯片流程

```
1. 选择要编辑的幻灯片（左侧缩略图）
2. 选择元素工具（顶部工具栏）
3. 在画布上点击或拖拽创建元素
4. 选中元素进行样式调整（右侧工具栏）
5. 使用快捷键或右键菜单进行快捷操作
```

### 10.3 演示流程

```
1. 点击"放映"按钮
2. 选择放映模式（全屏/演讲者视图/自动放映）
3. 使用翻页或标注工具
4. 演示结束后按 ESC 退出
```

## 12. 功能边界与限制

### 12.1 功能限制

| 功能 | 限制说明 |
|------|----------|
| PPTX 导入 | 约 60% 还原度，复杂动画和特殊效果可能丢失 |
| PPTX 导出 | 无法 100% 还原所有效果 |
| 移动端 | 仅支持基础编辑功能 |
| 多人协作 | 暂不支持实时协作 |

### 12.2 技术限制

| 限制 | 说明 |
|------|------|

| 字体 | 部分字体需要网络加载 |

### 12.3 数据限制

| 限制 | 说明 |
|------|------|
| 图片大小 | 建议不超过 10MB |
| 视频大小 | 建议不超过 100MB |
| 元素数量 | 单页建议不超过 100 个元素 |
